class TrialTransform¶

Attempt a transformation of particle position(s) or simulation box.

Inherits from Trial

Public Functions

TrialTransform(Pair *pair, Criteria *criteria, const char *transType)¶

Constructor

Parameters
  • transType: For rigid single-particle translations, “translate”. For rigid single-particle rotations, “rotate”. For “smart Monte Carlo, force biased”, use “smctrans”. Note that “smctrans” is for translational moves only, and not rotation, and these require center-of-mass forces. See http://dx.doi.org/10.1063/1.436415 For triclinic cells, “xyztilt”, “xztilt” and “yztilt”. For volume change, “vol” For x-dimension box length change, “lxmod”. Similarly “lymod”, “lzmod”.
