Position¶

class Position¶

Positions are a set of coordinates (abbreviated here as “coord”) in the Euclidean coordinate system. The number of coordinates is the dimensionality of the Euclidean space.

Subclassed by feasst::PositionSpherical

class PositionSpherical : public feasst::Position¶

Spherical coordinates are defined as follows: The first coordinate is rho >= 0. rho is the distance from origin. The second coordinate is theta. theta is the angle between x-axis and projection of line on x-y axis. The third and final coordinate is phi, 0 <= phi <= PI. phi is the angle between z-axis and line.
