ModelLJAlpha¶

class ModelLJAlpha : public feasst::ModelLJ¶

The Lennard-Jones potential, \(U_{LJ}\) is described in ModelLJ. In this class, the \(\alpha\) parameter may be varied (default: 6).

Subclassed by feasst::ModelLJCutShift, feasst::ModelLJForceShift

Public Functions

virtual void set_alpha(const double alpha = 6)¶

Set the value of \(\alpha\). The default value is 6.
