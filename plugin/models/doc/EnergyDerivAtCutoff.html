EnergyDerivAtCutoff¶

class EnergyDerivAtCutoff : public feasst::ModelParam¶
class ModelLJForceShift : public feasst::ModelLJAlpha¶

The Lennard-Jones potential, \(U_{LJ}\) is described in ModelLJ. This class implements the force shfited (FS) version which ensures both \(U(r_c)=0\) and \(\left.\frac{\partial U}{\partial r}\right|_{r=r_c}=0\).

\( U_{LJ}^{FS}(r) = \left\{ \begin{array}{lr} U_{LJ}(r) - U_{LJ}(r_c) - \left.\frac{\partial U}{\partial r}\right|_{r=r_c} (r - r_c) & : r < r_c \\ 0 & r \ge r_c \end{array} \right. \)
