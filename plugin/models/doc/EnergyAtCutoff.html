EnergyAtCutoff¶

class EnergyAtCutoff : public feasst::ModelParam¶
class ModelLJCutShift : public feasst::ModelLJAlpha¶

The Lennard-Jones potential, \(U_{LJ}\) is described in ModelLJ. This class implements the cut and shifted (CS) version which ensures \(U(r_c)=0\).

\( U_{LJ}^{CS}(r) = \left\{ \begin{array}{lr} U_{LJ}(r) - U_{LJ}(r_c) & : r < r_c \\ 0 & r \ge r_c \end{array} \right. \)

For a Weeks-Chandler-Anderson (WCA) potential, use this class. The cutoffs are computed by ModelLJCutShift::set_wca. Thus, one workflow is as follows:

ModelParams wca_params = configuration.model_params(); // copy model params ModelLJCutShift::set_wca(type1, type2, wca_params); // set cutoff wca->precompute(wca_params); // compute shifts, etc Potential::set_model_params(wca_params); // use wca_params.
