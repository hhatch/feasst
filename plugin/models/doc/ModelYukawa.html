ModelYukawa¶

class ModelYukawa : public feasst::ModelTwoBody¶

The screened Coulomb (Yukawa) potential is given by \( U = \epsilon \exp(-\kappa (r/\sigma - 1)) / (r/\sigma) \) Note that \(\kappa\) is dimensionless in this definition.
