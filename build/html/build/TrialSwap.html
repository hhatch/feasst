TrialSwap¶

class feasst::TrialSwap¶

Attempt to randomly swap the type of a molecule. As currently implemented, the position of the first particle in the molecule during swap is where the first particle in the other molecule is placed. Thus, for multi-site particles, the orientation is random.

Inherits from feasst::Trial
