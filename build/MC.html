void nMolSeek(const int nMol, const char *molType, long long maxAttempts, const double volumeExpansion = 1.2)¶

Parameters
  • nMol: number of molcules to seek
  • molType: If molType is empty (as typically used by clones), assume first type.
  • maxAttempts: maximum number of trial attempts
  • volumeExpansion: If pressure is set in criteria and seeking more particles, then begin expanding the volume by a factor of volumeExpansion, insert/delete particles, then perform a volume trial move at high pressure until original box size is obtained.

Attempt to seek nMol particles of type molType by inserting/deleting or particles. Note that detailed balance is not obeyed.
