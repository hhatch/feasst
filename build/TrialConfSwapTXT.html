class feasst::TrialConfSwapTXT¶

HWH: Note: This class is depreciated but remains as a stub file for HWH: Note: eventual support. Attempts configuration swaps between processors. These swaps may occur with variable beta and lnz, but not variable order parameter (e.g. nMol or pairOrder)
