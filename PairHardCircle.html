class PairHardCircle¶

Hard circles with implicit depletant model (Asakura-Oosawa) based on overlap of excluded area. \( U = U_{Hard} + \frac{\Delta A_{ex}}{\pi R_g^2} \phi k_B T\) The hard particle diameter is set to 1. by default (dCircle_).
