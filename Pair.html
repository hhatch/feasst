Pair¶

class Pair¶

Pair owns variables and subroutines associated with pair-wise interactions between particles. The pair class calculates the pair-wise interactions between atoms using an empirical, classical potential function for atoms. This class owns variables and functions associated with these interactions such as the forces, potential energy and virial (pressure).

In order to create a custom Pair, you can follow two similar procedures.

First, you may define a custom Pair code in the same file as “int main()” for C++. See an example of this for the Jagla potential test case.

Second, you may copy existing “pair_*” files and replace the class name and header guards (e.g. BASECLASS_DERIVED_H_).

Inherits from BaseRandom

Subclassed by PairHardCircle, PairHardSphere, PairHybrid, PairIdeal, PairLJ, PairLJCoulEwald, PairPatchKF, PairSoftSphere, PairSquareWell, PairTabular1D
