PairPatchKF¶

class PairPatchKF¶

Patchy orientational dependent square-well interactions. See the following publications: Fluid–fluid coexistence in colloidal systems with short-ranged strongly directional attraction Norbert Kern and Daan Frenkel J. Chem. Phys. 118, 9882 (2003); http://dx.doi.org/10.1063/1.1569473

As currently implemented, this class assumes only one central particle site with one or more patches, and not multiple particles, each with their own set of patches, bonded together.
